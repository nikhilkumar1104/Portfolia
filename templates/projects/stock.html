This project is a Stock Market Forecasting Dashboard built using Dash, Plotly, YFinance, and scikit-learn. The goal was to allow users to:

  • Search for any stock ticker
  • Visualize historical stock prices with Plotly graphs
  • Use moving averages and volume overlays
  • Predict future trends using ML algorithms

I integrated live stock data using the YFinance API and applied machine learning techniques like Linear Regression and Random Forest to visualize future price projections.
